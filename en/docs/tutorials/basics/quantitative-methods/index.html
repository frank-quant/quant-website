量化方法基础

量化交易的核心是使用数学和统计方法来分析市场数据并做出交易决策。

统计学基础​

描述性统计​

  • 均值：数据的平均值
  • 中位数：数据的中间值
  • 标准差：数据分散程度的度量
  • 偏度：数据分布的不对称性
  • 峰度：数据分布的尖峭程度

概率分布​

  • 正态分布：最常见的概率分布
  • 对数正态分布：适用于价格数据
  • t分布：小样本时的分布
  • F分布：方差分析的分布

时间序列分析​

平稳性​

时间序列的统计特性不随时间变化：

  • 均值平稳：期望值恒定
  • 方差平稳：方差恒定
  • 协方差平稳：自协方差只依赖于时间间隔

自相关​

时间序列与其滞后值之间的相关性：

import numpy as np
from statsmodels.stats.diagnostic import acf

# 计算自相关函数
autocorr = acf(price_data, nlags=20)

单位根检验​

检验时间序列是否具有单位根（非平稳）：

from statsmodels.tsa.stattools import adfuller

# ADF检验
result = adfuller(price_data)

技术指标​

趋势指标​

  1. 移动平均线（MA）

    • 简单移动平均线（SMA）
    • 指数移动平均线（EMA）
    • 加权移动平均线（WMA）

  2. MACD指标

    • 快线：12日EMA
    • 慢线：26日EMA
    • 信号线：9日EMA

震荡指标​

  1. RSI（相对强弱指数）

    • 范围：0-100
    • 超买：>70
    • 超卖：<30

  2. 随机指标（Stochastic）

    • %K线：当前价格在最高最低价中的位置
    • %D线：%K的移动平均

成交量指标​

  1. OBV（能量潮）

    • 累积成交量
    • 价格确认

  2. 成交量比率

    • 当前成交量与平均成交量的比值

机器学习方法​

监督学习​

  1. 分类问题

    • 预测价格涨跌
    • 算法：逻辑回归、SVM、随机森林

  2. 回归问题

    • 预测价格水平
    • 算法：线性回归、岭回归、Lasso

无监督学习​

  1. 聚类分析

    • 市场状态分类
    • 算法：K-means、DBSCAN

  2. 降维

    • 特征提取
    • 算法：PCA、t-SNE

深度学习​

  1. 神经网络

    • 多层感知机
    • 卷积神经网络（CNN）
    • 循环神经网络（RNN）

  2. 强化学习

    • Q-learning
    • 策略梯度
    • Actor-Critic

风险管理​

风险度量​

  1. VaR（风险价值）

    • 在给定置信水平下的最大损失
    • 计算方法：历史模拟、蒙特卡洛、参数法

  2. CVaR（条件风险价值）

    • 超过VaR的期望损失
    • 更保守的风险度量

投资组合理论​

  1. 马科维茨理论

    • 均值-方差优化
    • 有效前沿

  2. 资本资产定价模型（CAPM）

    • 系统性风险定价
    • Beta系数

实践建议​

数据预处理​

  1. 数据清洗

    • 处理缺失值
    • 异常值检测
    • 数据标准化

  2. 特征工程

    • 技术指标计算
    • 滞后特征
    • 交互特征

模型验证​

  1. 交叉验证

    • 时间序列交叉验证
    • 滚动窗口验证

  2. 样本外测试

    • 避免过拟合
    • 真实性能评估
