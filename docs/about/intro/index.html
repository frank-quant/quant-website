关于 Frank Quant

欢迎来到我的量化交易网站！这里是我分享量化交易知识、经验和实盘记录的地方。

个人简介​

我是一名量化交易爱好者，专注于算法交易和量化投资策略的研究与实践。通过这个网站，我希望能够：

  • 分享量化交易的学习心得
  • 记录实盘交易的经验教训
  • 提供实用的工具和代码
  • 与志同道合的朋友交流学习

专业背景​

教育经历​

  • 金融工程硕士：某知名大学
  • 计算机科学学士：编程和算法基础
  • 数学背景：统计学和概率论

工作经验​

  • 量化分析师：某对冲基金
  • 算法交易员：自营交易
  • 技术开发：金融科技公司

专业技能​

  • 编程语言：Python, C++, R, SQL
  • 量化工具：pandas, numpy, scipy, scikit-learn
  • 交易平台：Freqtrade, Backtrader, MetaTrader
  • 数据源：Yahoo Finance, Alpha Vantage, Binance API

交易理念​

核心原则​

  1. 数据驱动：基于客观数据和统计分析
  2. 风险优先：严格控制风险，保护资本
  3. 持续学习：不断优化策略和模型
  4. 长期思维：关注长期稳定收益

策略特点​

  • 多元化：不同市场环境下的适应性
  • 自动化：减少情绪干扰，提高执行效率
  • 可扩展：能够处理多种资产和策略
  • 可验证：通过回测和历史数据验证

实盘记录​

  • 绩效报告：定期的绩效评估
  • 经验总结：成功和失败的经验教训

书籍推荐​
